# 🏰 Clash of Clans Stats Viewer

## 📌 Project Overview
Clash of Clans Stats Viewer is a web application that allows users to explore real-time data from the Clash of Clans game using the official API. Users can search for player profiles or clans and view detailed statistics in an interactive and visually appealing interface.

---

## 🎯 Purpose
The purpose of this project is to:
- Learn and implement API integration using `fetch`
- Build a dynamic and responsive user interface
- Apply JavaScript concepts like array higher-order functions
- Create a useful tool for Clash of Clans players to analyze stats

---

## 🌐 API Used
- **Clash of Clans Developer API**  
  👉 https://developer.clashofclans.com/#/

This API provides:
- Player information (name, level, trophies, etc.)
- Clan details (members, war stats, clan level)
- Rankings and game-related data

---

## 🚀 Features

### ✅ Milestone 2 (API Integration)
- Fetch real-time data using `fetch()`
- Display player/clan information dynamically
- Loading indicator while fetching data
- Error handling for invalid inputs or failed requests
- Fully responsive UI

---

### 🔥 Milestone 3 (Core Features - Planned)
- 🔍 **Search**  
  Search players or clans using tags

- 🎯 **Filtering**  
  Filter clan members based on trophies or roles

- 🔃 **Sorting**  
  Sort players by trophies, level, or rank

- ⭐ **Favorites**  
  Mark and save favorite players/clans

- 🌙 **Dark Mode / Light Mode**  
  Toggle theme for better user experience
